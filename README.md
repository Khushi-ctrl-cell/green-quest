# 🌱 Green Quest

Green Quest is a sustainability-focused platform that encourages users to adopt eco-friendly habits through gamified experiences.

## 🚀 Features

* Sustainability challenges
* Environmental awareness
* Leaderboards
* Progress tracking

## 🛠 Tech Stack

React
JavaScript
Firebase

## 🎯 Mission

Encouraging sustainable living through interactive digital experiences.
